# Racing Russian Glossary

Use natural Russian motorsport language, not literal machine translation.

F1 examples:
- race pace -> гоночный темп
- qualifying pace -> темп в квалификации
- long run -> длинный отрезок
- stint -> стинт / отрезок
- dirty air -> грязный воздух
- clean air -> чистый воздух
- undercut -> андеркат / подрезка
- overcut -> оверкат
- power unit -> силовая установка
- floor upgrade -> обновление днища
- front wing -> переднее антикрыло
- rear wing -> заднее антикрыло
- parc ferme -> закрытый парк
- stewards -> стюарды
- race control -> дирекция гонки
- safety car -> сейфти-кар
- virtual safety car -> виртуальный сейфти-кар
- silly season -> трансферная возня / сезон слухов

NASCAR examples:
- car -> машина
- restart -> рестарт
- stage racing -> гонка по стадиям
- drafting -> слипстрим / драфтинг
- playoff bubble -> граница плей-офф
- crew chief -> крю-чиф
- pit road -> пит-роуд
- caution -> желтые флаги / caution

Style:
- Do not calque headlines.
- Keep names, teams, regulations, and track terms accurate.
- Rumors must stay rumors: "пишут", "по данным", "если подтвердится", "пока это уровень паддок-дыма".
